Automatización con scripts en Linux

La automatización con scripts es una técnica clave en Linux para simplificar tareas repetitivas, mejorar la eficiencia y administrar el sistema de manera más efectiva. Los scripts en Bash son los más utilizados, aunque también se pueden emplear otros lenguajes como Python o Perl.

1. ¿Qué es un script?

Un script es un archivo de texto que contiene una serie de comandos que pueden ejecutarse automáticamente sin intervención manual. Se utilizan para tareas como la gestión de archivos, la administración de sistemas y la ejecución de procesos en segundo plano.

2. Creación de un script en Bash

Para escribir un script en Bash, sigue estos pasos:

  1. Abre un editor de texto (Nano, Vim, etc.).
  2. Escribe el código del script, comenzando con la línea #!/bin/bash.
  3. Guarda el archivo con la extensión .sh.

Ejemplo de un script básico:

echo "Hola, bienvenido a la automatización en Linux"
fecha=$(date)
echo "La fecha y hora actual es: $fecha"

3. Asignar permisos de ejecución

Antes de ejecutar un script, es necesario asignarle permisos de ejecución:

chmod +x script.sh

Luego, puedes ejecutar el script con:

./script.sh

4. Variables y estructuras de control

Los scripts pueden usar variables y estructuras de control para hacerlos más dinámicos.

Uso de variables:

nombre="Eduardo"
echo "Hola, $nombre"

Condicionales (if):

#!/bin/bash
echo "Ingrese un número:"
read numero
if [ $numero -gt 10 ]; then
  echo "El número es mayor que 10"
else
  echo "El número es menor o igual a 10"
fi

Bucles (for):

for i in {1..5}
do
  echo "Iteración $i"
done

5. Automatización de tareas con cron

Los scripts pueden ejecutarse de forma automática con cron. Para editar las tareas programadas:

crontab -e

Ejemplo de tarea que ejecuta un script cada día a las 3 AM:

0 3 * * * /ruta/script.sh

6. Uso de scripts en administración del sistema

Los scripts se emplean para tareas como:

  • Copias de seguridad automáticas.
  • Monitorización de recursos del sistema.
  • Instalación y configuración de software.
  • Envío de alertas por correo electrónico.
